Feb 11, 2020

Over the course of the past 2 days I got started with laying out the plans for the app i'm going to be developing. I don't have any screenshots to share for this entry because none of the work done was done digitally. The plan is to create game pieces that have RFID tags embedded in them. 99% of modern Android phones have NFC, which can be used to read from and write to the 1KB memory contained within the RFID tags. The app will simply write a number into the tokens at the start of the game, and then load content from a pool based on what tokens are read from during gameplay. this will enable players to simply touch their phone onto the token to access dialog options.
